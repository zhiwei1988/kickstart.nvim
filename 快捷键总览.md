## 说明

- **Leader 键**: `<space>`
- **Local Leader 键**: `<space>`
- **基础快捷键**: 来自 Neovim 内置功能和 kickstart.nvim
- **插件快捷键**: 来自已安装的第三方插件

### 涉及的主要插件

1. **telescope.nvim** - 模糊查找和搜索
2. **nvim-lspconfig** - LSP 配置和支持
3. **neo-tree.nvim** - 文件浏览树
4. **gitsigns.nvim** - Git 行修改提示
5. **diffview.nvim** - Git diff 可视化
6. **nvim-dap** - 调试适配协议
7. **nvim-cmp** - 自动补全
8. **conform.nvim** - 代码格式化
9. **harpoon** - 快速文件导航
10. **marks.nvim** - 标记管理
